System Attributes

System Attributes are attributes (that is, information about assets and environments) defined by the system. System Attributes cannot be edited or deleted as they are integral to . The System Administrator can restrict permission to view System Attributes, apply formatting to alter the way each System Attribute is displayed, set whether historical values are recorded, and create additional statistical attributes for select System Attributes.

View System Attributes and Change Selections

  1. Navigate to Admin Console > Data Schema > System Attributes.
  2. Select a System Attribute from the list.
  3. If desired, update selections
  • Record Value Changes: Check this box if this attribute is required for historical reporting. If unchecked, retains only the current value of the attribute; no previous information is available. If checked, all value changes for this attribute for the life of the asset will be recorded for historical reporting and life cycle management.
  • Restrictable: Check this box if you would like this attribute to be one that can be limited to certain Groups

Group-based restrictions must be completed in User Access.

  1. If desired, apply Formatting changes to the Attribute that are triggered by specific values or ranges.
    1. Click Add to add a formatting change.


    2. In the Add Formatting dialog that appears, select foreground color, background color, operator, and value if appropriate. Click OK to return to the Formatting pane. Multiple conditions can be set for a single System Attribute.
  2. Click Save Changes when finished.

Create a New Statistical Attribute from a System Attribute

  1. Navigate to Admin Console > Data Schema > System Attributes and select an attribute to which you will apply a statistical view.
  2. Click the New Statistic button.

The selected attribute must generate numerical data. The New Statistic button is grayed out if statistical measurements are not relevant for the selected System Attribute.

  1. Complete all required and any desired additional fields for the new statistical attribute. Some fields are pre-populated with data inherited from the parent System Attribute.
  2. Apply Formatting if desired.
  3. Click Save Changes.
