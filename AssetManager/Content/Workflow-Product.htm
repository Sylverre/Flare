Introducing

The necessary processes for rolling out to your data center are summarized here. Click the links for more information and detailed instructions for each step.

  1. Deploy Hardware: prepare and physically install the hardware elements of the solution.
    1. Readers
    2. IR Locators
    3. Tags
  2. Install Software: create a database and install the software.
    1. Microsoft SQL Server or PostgreSQL database
    2. with Zone Manager
    3. Additional Zone Managers if required
  3. Configure System: associate the hardware and software and add licenses.
    1. Readers
    2. Tag Groups
    3. License Keys
    4. Serial Devices (if using)
    5. SMTP (if using)
  4. Populate System: define the locations to be monitored, types of assets to be managed, acceptable environmental conditions and system protocols for unacceptable environmental conditions.
    1. Location Hierarchy
    2. Rules
    3. Assets
    4. Summary Locations
    5. System Notifications
  5. Establish User Accounts: create user accounts, set permissions and restrictions.
    1. Create User Accounts
    2. Create Groups
    3. Optionally, integrate with LDAP/Active Directory
  6. Customize System: set up displays for different purposes and specific users.
    1. Dashboards
    2. Views
    3. Reports and Graphs
    4. Maps
    5. System Notifications

This manual includes brief, step-by-step directions for each stage of the process, and a final chapter on systems administration.  For more information, see the Systems Administrator manual or support.rfcode.com
