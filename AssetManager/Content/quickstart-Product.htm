Quickstart:

To use in your data center, you must:

  1. Deploy Hardware: prepare and physically install the hardware elements of the solution
  2. Install Software: create a database and install the software
  3. Configure System: associate the hardware and software and add licenses
  4. Set Up Your Properties: define the locations to be monitored, types of assets to be managed, acceptable environmental conditions and system protocols for unacceptable environmental conditions
  5. Establish User Accounts: create user accounts, set permissions and restrictions
  6. Customize System: create or edit dashboards, reports, and graphs

Each of these tasks is described in a chapter of this manual. More information is available online at support.rfcode.com
