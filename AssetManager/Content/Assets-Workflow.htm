Add Assets to

Assets are the physical objects that are managed and/or monitored with . You may choose to add assets that will not be tagged as well as those with tags not yet assigned. Assigning tags to assets can be done as part of, or separately from, the process of adding assets to . To manage assets that may be moved, assign a tag to an asset; to collect data from sensors, assign the sensors as assets.

Add a Single Asset

  1. From User Console > Assets click New to add an asset manually.
  2. Select the Asset Template or Asset Type from the drop-down and complete the Create: Asset dialog.

  1. When completed, click one of the buttons at the bottom of the dialog: Edit Group to restrict access to the asset type, Save & Add Another, or Save. Cancel closes the window without saving.

Assign Tag and Add Single Asset

  1. In User Console, navigate to Tag Management > Manage Tags.
  2. Select the tag(s) under the Import Detected Tags column and click the Add Selected Tag(s) button, or click Add All Tags. This will move the tags to the Unassigned Tags column.
  3. In the Unassigned Tags column, right-click on the tag to be assigned in the far right column and then select New Asset.
  4. Complete the Create: Asset dialog.
  5. When completed, click one of the buttons at the bottom of the dialog: Edit Group to restrict access to the asset type, Save & Add Another, or Save. Cancel closes the window without saving.

Add Asset Template

Create templates to pre-populate common asset attributes for asset types. Asset Templates minimize the need for data entry, in some cases leaving only Name and Tag to be completed by the user.

  1. Navigate to User Console > Customization > Asset Templates and click New.
  2. Complete the Template ID and Description fields, and select the desired asset type.
  3. Complete the asset type information fields that appear. Save Changes when finished. The template is now available in the User Console. To view or use, navigate to User Console > Assets > New and click the drop-down arrow for Asset Template.
