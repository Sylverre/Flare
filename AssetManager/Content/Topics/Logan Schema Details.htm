Schema Details

The standard schema included in defines assets and attributes for use in data center monitoring and management. New asset types and attributes created by a systems administrator are included in the schema once saved.

To view all items in the schema, navigate to Admin Console > Configuration > Export Configuration and Export Schema. The export file includes a description of each attribute or entity in the schema.

To view Asset Types in hierarchy, navigate to User Console > Asset Types and access the drop-down menu. Click the +/- buttons to expand or collapse categories.
