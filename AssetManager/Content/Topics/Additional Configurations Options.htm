Additional Configurations Options

The system.properties file contains configuration directives for the software that may not be set inside the database or may be needed in order to connect to the database when the service is started. Changes made to this file are preserved when is upgraded. The file is modified whenever database connection parameters are changed within the software. Additionally, directives can be modified and added by hand that will change the behavior of the software. It may be beneficial to have this file included in the regular backup regimen of the server. All other unique information for other than what is contained in this file is stored in the database*. The file is located at c:\program files\rfcode\\conf\system.properties or a similar path that was chosen to install the software.

Excludes OEM versions of and signed SSL configurations.
