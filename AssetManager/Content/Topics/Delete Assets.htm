Delete Assets

  1. Navigate to User Console > Assets. Using any preferred view or Filter, locate the asset to be deleted and click on the asset's name in the right pane. The Delete Selected Dialog box appears.

 

Select a tag-handling option:

  • Delete tag with the asset (tag is deleted with the asset and removed from the system)

    • Use this option if the tag will be retired or taken offline for a significant period.
  • Recycle the tag (tag is removed from the asset and moved to the unassigned tag list)

    Use this option if the tag will be affixed to a new asset.
  1. Click Delete Asset to continue.
