Zone Manager Configuration part 2 - 2012

Zone Managers may have been installed separately from , or as part of the installation.

  • Existing Remote Zone Manager: Zone Manager that you may have already utilized to connect with RF Code hardware prior to the release of the .
  • Local Zone Manager: Local Zone Manager that is automatically installed with the installation of Asset Manager (no configuration necessary).
  • New Remote Zone Manager: If you have a larger deployment scenario, you may be installing remote Zone Managers to interact with your .

To configure to interact with a newly installed remote Zone Manager, follow the steps outlined below. To “integrate previously installed and running Zone Managers, please refer to the Remote Zone Manager Integration section of this document.

Navigate to Configuration > Zone Managers. The Zone Manager configuration screen will appear.

Input the information for the Zone Manager(s) you want to connect with. To do this, click the New button and input the required information (see below). Click . The Zone Manager you have just configured will appear in the panel to the left.

Zone Manager Information Input Task Pane: Basic Information
  • Name: Create a name for the Zone Manager.
  • Description: Enter a description for this Zone Manager.
  • Enabled: If you would like your Zone Manager to go active and receive and transmit tag data once you configure it within this box should be checked. Zone Manager Information Input Task Pane: Network Settings
  • Hostname: Enter the IP address of your Zone Manager in this field.
  • Port: Enter the port number over which to communicate with your Zone Manager (by default this is 6580).
  • SSL: If you would like to use SSL settings with your Zone Manager, select one of the options from the drop down list (Do not use SSL, SSL - No Verification, SSL Verify Certificate & Hostname). (Please refer to the Configuring SSL for Asset Manager and Zone Manager section of this manual for more information).
Zone Manager Information Input Task Pane: Up Connect Settings
  • Up Connection Zone Manager ID: This ID uniquely identifies the Zone Manager to the (defaults to RFCodeZoneMgr).
  • Up Connection User ID: This ID is the user ID used to authenticate with the remote Asset Manager.
  • Up Connection Password/Confirmation: If you assigned a password when you configured your Up-Connection Zone Manager, you will need to enter and confirm the password for use with .

The feature allows for Zone Manager communication with Asset Managers to be more “firewall friendly”. When this option is configured the Zone Manager to initiates the contact with the Asset Manager rather than requiring that access be granted to allow the Asset Manager to connect to Zone Managers behind a firewall. Up Connection settings must be specified in the configuration of your remote Zone Manager prior to configuring this Zone Manager for use with Asset Manager. The Up Connection Zone Manager ID and password are obtained in the configuration of the remote Zone Manager. This feature is only available for Zone Managers that have been upgraded to version 1.4 or higher.

Zone Manager Information Input Task Pane: User Account
  • Adminstrator Username/Password: This ID and password is assigned automatically by Asset Manager. The default ID is admin and the password is a randomly assigned password known only to the . If you would like to change this setting, type in an administrator username and password of your choosing. This will then be used to access your Zone Manager or Remote Zone Manager for configuration purposes.
  • Read-only Username/Password Input the username and password if you have one configured for use with your Zone Manager that limits the Zone Manager to read-only access restriction.
Zone Manager Information Input Task Pane: Event Store & Forward
  • Maximum Event Storage: Input the amount of historical data to be stored in hours in the event that the connection to the is interrupted.

Remote Zone Managers have the capability to store (on a local hard drive) tag events if the connection to the Asset Manager is interrupted and then forward the stored events when the connection is restored. This capability is only limited by the amount of hard drive space.

Zone Manager Information Input Task Pane: Advanced Zone Manager Settings
  • Offline Timeout: Specify the amount of time that you want to pass before an offline Zone Manager is reported as offline by .
  • Expected Location Bias: Specify the confidence factor that you would like added to the expected locations rule.
