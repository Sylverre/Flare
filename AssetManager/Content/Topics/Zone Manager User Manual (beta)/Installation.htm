Installation

Windows Installation

To install the Windows version of Zone Manager, the user must browse to the setup.exe file in the Windows folder of the Zone Manager CD. Once this file has been located the user should double-click on it and follow the installation wizard prompts to choose a location to install the program files to. Once complete the user can then browse to the application’s README document located in the Start Menu to familiarize themselves with the program.

Linux Installation

To install the Linux version of Zone Manager use the following command:

rpm –ivh rfcode-zonemgr-1.0.0-20071207_152357_P.noarch.rpm

This command will install the software and start the server. The server will be automatically configured to start at boot as well

The name of the rpm may be different depending on what version of the program you are installing

If you are upgrading from previous version of Zone Manager use the following command:

rpm –Uvh rfcode-zonemgr-1.0.0-20071212_153937_P.noarch
