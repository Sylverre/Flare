Requirements

The Zone Manager 1.0.0 CD-ROM contains several items to assist partners and customers with using the Ranger software with R200 readers:

  • Windows installer executable
  • 32 bit Linux RPM
  • 64 bit Linux RPM
  • TAR.GZ file non-standard Linux installs
  • User Manual - end user documentation for Zone Manager 1.0.0
  • API Reference Document - a specification and code samples for 3rd-party RFID application development

Introduction

Zone Manager 1.0.0 is a software engine that is designed to accompany the R200 Reader to provide accurate, rack-level inventory of IT assets in real-time. Ranger is a command line application/daemon that allows customers to build applications to accomplish rack-level inventory using the R200 hardware. The purpose of Ranger is to aggregate data from all defined readers/racks and via an API interface and return the list of tags IDs in an individual rack, groups of racks or all racks defined. Ranger is an intelligent multiplexer reader interface with a simple telnet style API (command–response) interface.

Requirements

Zone Manager Server:

  • Pentium Dual-Core or Athlon-Class Processor
  • 1GB RAM
  • 300MB Free Disk Space
  • Windows XP, Service Pack 2
  • RedHat Enterprise Linux 5 (however, has not been tested yet)
  • Fedora 8

NOTE: Administrator access on Windows or root access on Linux is required

Zone Manager Client:

  • Internet Explorer 6, 7
  • Firefox 1.5, 2

Communications Ports

Telnet:

6501 – command & response interface

6502 – tag change events interface

Web Port:

6580 – command & response interface
