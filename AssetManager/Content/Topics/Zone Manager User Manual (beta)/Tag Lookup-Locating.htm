Tag Lookup/Locating

The following are a list of commands to use the Zone Manager software to lookup tag data:

Tag Print

tagprint <tag-guid> ...

http://<baseurl>/tagprint?tagid0=<tag-id>&tagid1=<tag-id>&...

where <tag-guid> is the GUID of a specific tag

This function prints details of one or more specific tags, including all tag attributes and all the tag’s active tag links (including SSI values).

EXAMPLE COMMAND

tagprint RFCRCK00000051

OR

http://10.1.20.90:6580/rfcode_zonemgr/zonemgr/api/

tagprint?tagid0=RFCRCK00000051

http://10.1.20.90:6580/rfcode_zonemgr/zonemgr/api/tagprint.

xml?tagid0=RFCRCK00000051
