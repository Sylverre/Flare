Reporting Tag Data

The following are a list of commands to use the Zone Manager software to report tag data:

Tag List

taglist

http://<baseurl>/taglist

This function lists all tags currently being observed, including tag ID and tag group ID.

EXAMPLE COMMAND

taglist, taglist.xml

OR

http://10.1.20.90:6580/rfcode_zonemgr/zonemgr/api/taglist

Tag Count

tagcount

http://<baseurl>/tagcount

This function returns the current number of all tags being observed.

EXAMPLE COMMAND

tagcount

OR

http://10.1.20.90:6580/rfcode_zonemgr/zonemgr/api/tagcount

EXAMPLE RESPONSE

TagCount: count=9

#

OR

TagCount: count=9

Introduction

Tag List by Location

taglistbylocation <loc-id> ...

http://<baseurl>/taglistbylocation?locid0=<loc-id>&locid1=<loc-id>&...

where <loc-id> is alphanumeric ID for one or more specific locations (zones).

This function lists all the tags found in one of the listed locations (zones), or any of their children. It also

returns tag IDs, as well as all location attributes for the zone where the tag is found.

EXAMPLE COMMAND

taglistbylocation Rack1

OR

http://10.1.20.90:6580/rfcode_zonemgr/zonemgr/api/taglistbylocation?id=Rack1,

http://10.1.20.92:6580/rfcode_xonemgr/zonemgr/api/taglistbylocation.xml?id=Rack1

EXAMPLE RESPONSE

country=”USA”, state=”Texas”, suite=”500”

Tag Count by Location

tagcountbylocation <loc-id>

http://<baseurl>/tagcountbylocation?locid0=<loc-id>&locid1=<loc-id>&...

where <loc-id> is alphanumeric ID for one or more specific locations (zones)

This function returns count of tags contained in one or more specific lcations, or any of their children.

EXAMPLE COMMAND

tagcountbylocation Rack1

EXAMPLE RESPONSE

TagCount: count=3

Tag Print by Channel

tagprintbychannel <reader-id> ...

http://<baseurl>/tagprintbychannel?channelid0=<reader-id>&channelid1=<reader-id>...

where <reader-id> is the alphanumeric ID for the reader to be queried

This function prints details of all the tags currently being observed by any of the channels of specific readers.

EXAMPLE COMMAND

tagprintbychannel R60

OR

http://10.1.20.92:6580/rfcode_zonemgr/zonemgr/api/tagprintbychannel?channelid0=

R60&channelid1=R60

http://10.1.20.92:6580/rfcode_zonemgr/zonemgr/api/tagprintbychannel.xml?channeli

d0=R60&channelid1=R60
