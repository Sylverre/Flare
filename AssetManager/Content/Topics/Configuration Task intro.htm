Configuration Task Pane

Configuration tasks are available from the Administrator Console > Configuration task pane.

 

The five basic components listed in System Configuration must be configured before can be used to manage assets and monitor environmental conditions. Other configuration subtasks can be completed at any time.

Configuration Subtask Purpose
Database requires a database to function. All configuration and information is stored in this database.
License Keys A basic license key is included with your purchase. Extra license keys are available to expand your system to incorporate more racks or for software integration module entitlement. Contact your Representative or reseller for more information.
SMTP Server An SMTP server must be configured for to email alerts and reports.
Import Configuration Importing tag group, reader, location, rule or system configuration allows administrators to populate or restore a system without manual entry. This sub-task allows administrators to import previously saved configurations from an existing external Zone Manager or .
Export Configuration This sub-task allows administrators to export configuration information. The main purpose of this feature is to modify multiple attributes at the same time, rather than using the web interface. Exported configurations can also be used to replicate a configuration on an alternate system.
Tag Groups Tag Groups must be set up before tag data can be interpreted.
Sensor Definitions If using Current Loop or Voltage Sensors, set mA or V readings.
Zone Managers

A Local Zone Manager is automatically installed with . Depending on your deployment scenario, you may require additional Zone Managers. From the Configuration task, you can add or delete Zone Managers.
Zone Manager Status This sub-task provides a read-only summary view of your current Zone Manager(s) status.
Readers Allows administrators to configure readers for use with . There are two task panes. On the left is the tree of installed readers and on the right is the task pane where reader configuration information is entered.
Reader Firmware has a Reader Firmware upgrade feature available that will allow for upgrading individual or multiple RF Code Readers at once. To upgrade an RF Code Reader’s firmware using , navigate to Configuration > Reader Firmware. The most up-to-date version of reader firmware is bundled with the software at the time of release.
Reader Status Provides a read-only summary view of your current reader(s) status.
GPS Enables adjustment of the global GPS settings for all GPS devices that are connected to the system.
Serial Devices Allows administrators to configure connected serial devices to receive alert actions from an Reader for open and resolved threshold conditions.
Server Allows administrators to set the units of measurement that will record to the database when various calculated attributes are performed, and to adjust the job schedule.
