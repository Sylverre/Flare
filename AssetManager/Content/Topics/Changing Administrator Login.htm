Changing Administrator Login

To change the Admin password, perform the following steps:

  1. Go to the Admin Console.
  2. Click on the User Access tab in the left column and then click on Users.
  3. In the column to the right of the User tab, click on the user named admin.

  1. In the User Information field, type the new password in the Password and Confirm Password fields.

Record the new password someplace secure.

  1. Click the Save Changes button.
