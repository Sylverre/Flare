update(target attribute, value) or update(asset reference, target attribute, value)

When an expression involves multiple statements, the expressions are evaluated one at a time, from left to right.  The last statement which is evaluated will be returned as the value of the calculated field attribute. If any exception occurs anywhere in the evaluation process, null is returned as the value of the calculated field.
