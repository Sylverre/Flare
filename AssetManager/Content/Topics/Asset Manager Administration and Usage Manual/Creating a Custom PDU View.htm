Creating a Custom PDU View

Because the Default View does not contain columns to display values for standard PDU power Attributes, you will want to create a new view to show these at quick glance.

For more about Views, refer to the View section in this guide.

To create a custom PDU View, perform the following steps:

  1. Go to Customization > Views and click the New button create a new view to show specific power attributes.


  2. Configure the View to show the Attributes you want to see and choose which User Groups will have access to it.


    The new view appear in the list of Views.


  3. To see the PDUs and the values for their power-related Attributes, go to Assets > Manage Assets and choose the new view from the View drop-down menu.


Depending on the number of Assets in your system, you may have too many Assets listed to see any PDUs on the screen and the PDU Attributes in the PDU View may not be relevant.  To shorten the list, restrict the list with the Type filter to show only PDUs.

After being restricted by the View and the Filter, the Manage Assets list will then show only PDU assets. Use the View drop-down if desired to change views of these assets.
