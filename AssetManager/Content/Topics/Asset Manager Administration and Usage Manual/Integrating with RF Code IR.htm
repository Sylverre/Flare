Integrating with IR Locators

When you deploy an IR locator, you need to configure it both through its configuration utility and also in .  Rack Locators and Room Locators are both IR locators and they each have their own configuration utility. 

When installed, IR locators send location beacons, which are detected by IR-enabled tags. These tags include the received IR beacons in their transmissions to the readers.

To configure an IR Locator, perform the following steps:

  1. Ensure that at least one reader has been configured.

For more information, refer to the reader configuration sections:
Reader Configuration with the Reader Web Interface
Adding and Configuring Readers in the Admin Console

  1. Add the right IR Tag Groups for your particular IR tags to .

For more information on adding Tag Groups, refer to the Adding Tag Groups section.

  1. Configure the IR Locator with its specific locator configuration utility.

For more information, refer to the user guide for that locator:
http://support.rfcode.com/customer/portal/articles/722910

  1. Associate an IR Rule to a Location in .

For more information, refer to the Locations and Rules section.
