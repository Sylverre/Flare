Blocking HTTP Access in

To configure to block HTTP access from external requests, perform the following steps:

  1. From the host running , go to Control Panel > Administrative Tools > Services and stop the service.
  2. Edit the System Properties file in { Install Path}\conf directory by removing the line “http.port=6580”
  3. Save the file.
  4. Re-start the service.

    Now, only localhost can get to port 6580. A request from an external host cannot.
