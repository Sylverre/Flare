Log Files

The Log Files section of the Appendix contains information about finding and viewing log files.  The log files will contain details about how the code is being executed and why certain tasks might be performing in unexpected ways or not at all. 

RF Code logs are and have been part of the HTTP API since version 2.5.  In the links below, “IP Address of ” is the IP address of the application server.

 For versions of earlier than 2.5, the logs can only be found at the following file path on the server:  \Program Files\\\jetty\logs and will look like the following:


Log files can be downloaded at the following URL:
 http://<IP Address of >:6580/api/system/files

Log files are organized by date in the form of year_month_day.stderrout.log

To change the logging level for various modules within , go to the following URL:
http://<IP Address of >:6580/api/system/log

Statistics regarding the log level can be viewed at:
http://<IP Address of >:6580/api/system/stats

 

Contact Support for assistance if you need to review log files in order to troubleshoot an issue.
