BACnet Slave Object IDs

For each attribute value to be published by the BACnet Slave Devices, will automatically assign a unique BACnet object ID used by BACnet clients to query an attribute’s current value. The BACnet Slave Object IDs panel allows you to view the value of the BACnet object IDs has assigned to each attribute published via BACnet.

  1. Navigate to Admin Console > Integration > BACnet Slave Object IDs.
    A list of the currently defined BACnet Slave Devices appears in the right-hand pane.
  2. Select a slave device to view the BACnet object IDs has assigned to each attribute value.
    There may be several Object IDs depending on the type and number of attributes that you specified.


  1. The entire BACnet Object ID attribute mapping for a BACnet Slave Device can be exported in XML, CSV, or PDF format to allow a BACnet administrator to conveniently configure BACnet client software. To export these Object IDs use the Export XML, Export CSV or Export PDF buttons.
  2. Access your BACnet system to configure your BACnet client software and complete the integration.

For information regarding how to integrate your BACnet system please refer to your BACnet client manuals or refer to the BACnet website http://www.bacnet.org.
