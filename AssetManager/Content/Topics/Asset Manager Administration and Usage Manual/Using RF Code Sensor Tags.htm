Using Sensor Tags with PDUs and CDUs

The exact process of physically deploying and configuring PDUs and CDUs for use with depends on the make of the power device(s) you are using, but essentially the process is the following:

  1. Physically connect the R170 sensor tag.
  2. Add the Tag Group and Sensor Tag Asset to .
  3. Modify a custom view so you can view power attributes for the PDU or CDU.
