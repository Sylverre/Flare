Modbus Slave Server

To enable Modbus integration and to set the Port, perform the following steps:

  1. Navigate to Admin Console > Integration   > Modbus Slave Server and select the Enabled check box.

  2. Under Modbus Slave Server Configuration, set the Port.

By default, the port is set to 502. However, this is a common port used for networking industrial electronic devices. If you complete the configuration of for Modbus and there is still no communication between the and BMS systems (for example, the Port reports that it is not responding), change this port setting (to some port higher than 1024, such as 9502) and test communication between the systems again.

Ensure the port used is open in your firewall, whether you use port 502 or another port, to allow the BMS/Modbus Server access to the data.

  1. Save Changes.
