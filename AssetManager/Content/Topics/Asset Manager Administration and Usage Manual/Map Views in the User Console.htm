Map Views in the User Console

Map Views are available to different Users in the User Console depending on how they are configured in the Admin Console.

Use the drop-down menus to select the desired Location, Map, View, Attribute, Zoom, Alert Severity, and/or Attribute Formatting.

  • Location
  • Map
  • View
  • Attribute
  • Zoom
  • Minimum Alert Severity
  • Attribute Formatting

Using Map Views, the mapʼs creator, Systems Administrator, or another user with sufficient access can change which Attribute will be shown for any particular Hot Spot and whether or not the information will be shown to users by default or whether the information will remain hidden until a user hovers over it with the mouse pointer.
