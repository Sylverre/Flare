JMX Monitor

This sub-task lets you publish data using Java Management Extensions (JMX).  To use this feature, you will need a separate license to enable the functionality. Once the license key has been installed, JMX publishing is done through the configuration of the sub-tasks JMX Monitor and JMX Domains.

When configuring the JMX Monitor you will need to specify the hostname and port of the computer where the JMX software is installed. defaults to the local host. The local host should be used if the software is installed and running on the same computer as .

To configure to use JMX, perform the following steps:

  1. Install the JMX License Key using the same steps as outlined in the License Key Configuration sub-task.
    The JMX License Key is a separate key that must be installed in addition to the license key. Once the key has been properly installed, it will appear in the license key list.
  2. After you install the JMX key, restart the service via the Microsoft Windows Service Manager or reboot the application server.

If you are running in a Linux environment and need more information about how to restart the service, refer to Linux Installation section of this document.
