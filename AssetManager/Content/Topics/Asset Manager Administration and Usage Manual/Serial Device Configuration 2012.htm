Serial Devices Serial Device Configuration

If alerts are to be sent to third-party serial devices, those devices must be connected to Readers (typically by USB cable) and added to the system. Once the serial devices are connected, alert actions can be created to send messages to the serial devices for open and resolved threshold conditions. The Serial Devices Configuration has two task panes. The left displays the list of readers that have been enabled for Serial Device communication (See Reader Configuration section). The right task pane is for the reader configuration information.

Configure a Serial Device

  1. Ensure that your serial device is physically attached to the Reader and that the Reader has been configured properly to allow for serial device communication.
  2. Select the serial device and Reader from the list in the middle pane.
  3. Input the information for your serial device.
    1. Name: Enter or modify a name for the serial device.
    2. Allowed User Groups: Click the ... button for this field and a selection box will appear. Select the user groups that you would like to allow access to this serial device if you have enabled the Asset Security feature (See Server section).
  4. Click the Save Changes button.
