The User Console

The User Console lets Users manage assets and monitor environmental conditions, whether the system is deployed in a data center or elsewhere. The User Console Task Pane includes:

  • Dashboard
  • Tag Management
  • Assets
  • Customization
  • Maps
  • Reports/Graphs
  • Events
  • Alert Management

All of these tasks are viewable to users of any role, but roles determine which users have permission to create, edit, or delete items within each task.

For screenshots and brief descriptions of all the Tasks and Sub-Tasks available in both the Admin Console and the User Console, refer to the Admin Console and User Console Task Overview Matrix section in the Appendix.
