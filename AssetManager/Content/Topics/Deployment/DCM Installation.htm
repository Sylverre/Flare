Installation

If you have selected the database to be used, set up the database before installing . Otherwise, for a pilot deployment, install using the Microsoft SQL Server Express that is bundled with the product.

Per database requirements, choose the SQL Server Express option only for pilot installations, not for full roll-outs when you deploy into a production environment.

To install :

  1. Run the installer image provided by or insert the CD.
  2. Follow the prompts of the Installation Wizard.
  3. Choose the Local Zone Manager option.
    •If you are using an external database, do not check the Install and Configure Microsoft SQL Server Express box. You will later be able to specify the database to which will connect.
    •If you are using the default SQL Server Express Database that is integrated into the installer, check the Install and Configure Microsoft SQL Server Express box.

If you are using the SQL Server Express option, do not change the default SQL password assigned to the sa administrator account.

  1. The Admin Console will launch automatically upon completion of installation.

It will take a few minutes for Zone Manager to initialize.

  1. Log in with the default Administrator user name and password (case sensitive):
  • Username: admin
  • Password: admin
  1. After logging in, proceed to the Software Configuration section(s).

At any time, you can launch the web interface by navigating to Start > Programs > > or by navigating in a browser to http://{server host name}:6580

Refer to the System Administration Guide for further details about installing and configuring . This is available from Support.RFCode.com.
