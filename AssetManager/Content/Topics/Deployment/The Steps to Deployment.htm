The Steps to Deployment

The necessary processes for completing a pilot deployment are summarized here. Click the links for more information and detailed instructions for each step.

  1. Install Software in Windows or Linux.
    1. Run the Installer Image EXE provided by or insert the CD-ROM.
    2. Follow the prompts of the Install Wizard.
    3. Choose the Local Zone Manager option.
    4. Check the Install and Configure Microsoft SQL Server Express box.
    5. The web interface launches automatically upon completion of installation. This can take a few minutes.
      • If launch seems to be taking too long, navigate to Start > Programs > RF Code > or, in your browser, to http://<server host name>:6580
    6. Log in with the default Administrator user name and password (login is case sensitive):
      • username: admin
      • password: admin
  2. Configure and mount IR Locators and Readers
    1. Install appropriate configuration utility from or CD-ROM to a Windows computer.
    2. Plug reader or locator in via USB 2.0 cable.
    3. Follow Install Wizard prompts and directions to configure device(s).
    4. Mount locators and readers in data center.

  3. Add Readers to
    1. Log in to .
    2. Log in to with the default Administrator user name and password (login is case sensitive):
      • username: admin
      • password: admin
    1. Navigate to Configuration > Readers.
    2. Click New and select the reader type from the drop-down list.
      • Name: create a name for the reader.
      • Hostname: enter the IP address of the reader.
      • UserID/password: if you have enabled user access on the reader, enter the user ID and password.
    3. Save Changes.



    If configuring an Up Connection Reader, you must specify the reader Up Connection ID that was assigned to that reader and also, if specified, the optional password from the Reader Configuration Utility. For more information, see The Reader Configuration Utility User Manual

  4. Configure Tag Groups

    To test reader and tag functionality, add one or more tag groups.

    1. Navigate to Configuration > Tag Groups.
    2. Click New and select a treatment code.
    3. Enter a name for the tag group. Update Group Code to match label if required.
    4. Save Changes.

  5. Add Licenses
    1. Navigate to Configuration > License Keys.
    2. Click Add License Key and add your 16-digit license key code(s) from the License Card.
    3. The System Configuration pane indicates that the System Configuration has been completed.

  6. Add Assets to the System

    To confirm functionality, add a few sample assets.

    1. Navigate to User Console > Assets > Manage Assets.
    2. Click New and select an asset template or asset type from the drop-down list.
    3. Complete all required and any desired additional fields.

    In a pilot deployment, the location drop-down list may not be populated beyond the top-level Location. In that case, select Location or leave the field blank. Do not type in a location name. A Location Hierarchy must be created for use in the production environment, and manual entry of locations can interfere with this process.

    1. Click Save.
