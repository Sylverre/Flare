Consoles

There are two primary views (consoles) in , accessible at any time:

  • Admin Console: Access this console by clicking the Admin Console link in the lower right corner of the web interface of . This console is typically used for administrative functions of the server and where a large portion of this document focuses.This is where most of the initial configuration is done.
  • User Console: Access this console by clicking the User Console link in the lower right corner of the web interface of . This console is for managing and viewing sensors, alerts, dashboards, reports, graphs, etc. This is where the final configuration and daily use of is done.

Switching between the two consoles is done by clicking either the Admin Console or the User Console link at the bottom right of the user interface. The two links toggle depending on which console you are using.
