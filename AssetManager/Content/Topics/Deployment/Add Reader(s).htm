To add readers to, go to Admin Console > Configuration > Readers, click New, and then select the M250 reader type.

Populate the following reader configuration fields seen and described below.

  • Basic Information
    • Name: This is the identifier that will be seen in ; it is a text field, and is not the hostname or IP of the Reader.
    • Zone Manager: By default it will use the embedded Zone Manager that is installed with . This will not need to be changed unless you have multiple data center sites where it is recommended that you setup a Zone Manager instance for each site.
  • Network Settings
    • Hostname: This is the Hostname/IP Address of the Server
    • Port: By default this is set to 6500, and will most likely not need to change. This is the TCP port on the /Zone Manager Server that will accept connections from the Reader(s)
    • SSL Mode: This defines the encryption level, SSL that is used between the Reader and /Zone Manager, again the default should suffice.

    Authentication: If you chose to set a username/password on the reader in the Reader Configuration earlier, these fields must match the Reader’s credentials.

  • Up Connect Settings: If you chose an alternate configuration for your reader(s), such as using DHCP and Up Connect, then enter the same information in these fields as well as they must match precisely in order to allow the reader to connect to the and Zone Manager.

NOTE: For more information about Up Connect, refer to the following KB article: http://support.rfcode.com/customer/portal/articles/744747
