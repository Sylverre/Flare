Database Requirements

In addition to the server, a database is required for storing sensor tag data and configuration data that is gathered and managed by . If you have an existing database environment, create a new database and then connect it to . If you don’t have a database environment prepared or available or you want the quickest way to dive into your pilot installation, you can use Microsoft SQL Express 2008, which is bundled with . However, this “lite” database is not recommended for long-term or production use; therefore, if you do install with SQL Express, then you will need to migrate to a production-quality database during the full roll-out by following standard database backup and restore procedures or by using the export and import functions. For more information about exporting configuration settings, refer to the Administrator’s Guide online.
