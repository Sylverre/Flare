Configuring M250 Readers via USB

The Reader Configuration Utility (RCU) must be installed and readers configured via USB during initial deployment. The RCU is available on the CD included with the reader, and is available for download from support.rfcode.com.

All readers should be configured, if necessary, prior to being installed in their permanent locations. Readers must be configured prior to use if a static IP address needs to be assigned or if the Up Connect feature is to be used. Readers other than the first may not need to be pre-configured if the intent is to use DHCP for IP address assignment.

Configuring the Reader for Static IP Address or DHCP with Up Connect

  1. Install the RCU on the PC being used to configure the reader. If installing from CD, the CD should remain in the CD-ROM drive during the install.

If you run into any issues installing the driver, refer to the following KB Article: http://support.rfcode.com/customer/portal/articles/721801

  1. Connect the reader to the USB port using the USA-to-USB cable included in pilot pack.

USB cables are not provided with purchases other than pilot packs; they can be purchased at an electronics store or big box general retailer.

  1. When prompted, install the RFCUSB.inf driver from the configuration utility folder of the CD-ROM or installation directory from the download.
  1. Run the configuration utility.
    1. Select Local as the connection method. The M250 Reader should be seen on one of the COM ports listed.
    2. Select the COM port and click Next.
    3. If desired, change any setting in the configuration tabs including the IP address and Up Connect settings.
    4. Click Save Changes so that the configuration is saved to the reader.
  1. After a static IP is entered, you will be able to connect to the M250 Reader with a web browser. If you wish to use DHCP, the Reader Configuration Utility will also report the IP address that has been assigned to the reader. However, this address may change. Therefore, if you choose this manner of configuration instead, use the Up Connect feature to connect the M250 Reader to .

For more information, refer to the M250 Reader User Manual available at http://support.rfcode.com/customer/portal/articles/722910-user-guides-manuals-docs.

  1. Following configuration, attach the two helical antennas packaged with the reader. Align the antennas at a right angle from one another.

A reader configuration file may be provided by and loaded onto a USB thumb drive, created following the instructions in the M250 API Specification document, or exported from following the pilot deployment, and used to configure readers for a full roll-out. Plug the USB containing the CSV into the reader and it will begin processing. The USB drive is safe to remove as soon as the LEDs begin flashing.
