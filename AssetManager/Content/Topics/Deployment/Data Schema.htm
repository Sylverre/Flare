A data schema is the combination of asset types and attributes that allow you to determine what types of assets/sensors you are monitoring and specific attributes related to those. By default, contains two schemas that can be used in a general environmental monitoring deployment. Custom schemas can also be imported. For both the pilot and roll-out, you will use one of the default schemas. If you have specific needs in the future that are not addressed by the default schema, please consult with Support to customize the default schema or design a schema from scratch. Depending on the nature of the customization, it may also be necessary to work with Professional Services.

Import a Default Schema

  1. From the Dashboard/Admin View, select Schemas and Schema Import.

figcaption

  1. Click a schema name in the right-hand pane and click the Schema Description button to view details.
  2. Click Load Schema to add the described asset types, attributes, and hierarchy.

NOTE: For more information about Asset Types, refer to the following KB article: http://support.rfcode.com/customer/portal/articles/977425-how-to-create-new-asset-types-in-asset-manager-and-sensor-manager
