Editing Maps

Editing a Saved Map’s Settings

If you would like to edit the main settings for a Map that were originally specified when the map was first configured, click the Edit Map button and the Map Settings screen will appear. Change any settings you would like and click OK to save these changes.

Copying a Map
  1. If you desire to have maps that are similar to another map that you have already configured with a few minor adjustments you can use the Copy Maps button to do this. Select the map from the location tree that you would like to copy and select the Copy Maps button. The Map Settings screen will appear and at a minimum you are required to change the Map Family to save a copy of the map. Name is required to be unique so the name copied from the original map needs to be changed You can also change any of the map settings available in this screen. Click OK to save a copy.
  2. Once this copy has been saved, you can edit or add any of the Hot Spots or Map Attribute Boxes that you desire and click the Save Changes button to save.
Previewing a Map
  1. While configuring a map in the Map Configuration sub-task, you can preview how this map will look in the User Console by selecting the Preview Map button. This will open the map in a separate tab of your browser and will show how the map will appear when accessed in the User Console.
Deleting a Map

To delete a map, select the map you intend to delete from the locations tree and click the Delete Map button. You will be prompted by a dialog box asking if you are sure that you want to delete this map. Click the Delete button. The map will now be removed from the location tree.

Changing Global Settings

Global Settings applies to the fill color and opacity features of all maps that the display properties have not been configured yet. Select a fill color from the provided palette and an opacity level. The lower the percentage the more transparent the fill area will appear. For example a specification of 100% for this field will make the fill area completely opaque. If you choose not to specify a percentage for this field, the hot spot fill will appear 30% opaque. To make the fill color disappear entirely, choose 0% for this setting. Click OK to save the changes.
