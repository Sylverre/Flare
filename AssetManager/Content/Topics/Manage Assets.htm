Manage Assets

 

  • Click an Asset Link from the Assets menu to view a filtered list of assets in the right pane.

If multiple Asset Link Views are available, select a view from the View drop-down in the upper right of the window, beneath the logo.

  • Sort an asset list by clicking the column header.
  • Filter an asset list by selecting the desired filter(s) (Type, Location, etc.) and clicking Go.
  • Select an asset from the list by clicking the asset name. After selecting an asset,
    • Double-click to view information,
    • Click an action button from the toolbar, or
    • Right-click for additional options.

Manage Equipment with Unresolved Locations

 

Click this link to view tagged assets detected in locations to which they are not assigned. Double-click an asset from the list to view its attributes. Right-click the asset for more options.

A tagged Asset may have an Assigned Location, an Assigned To... summary location, and a Detected Location. The Assigned Location and Assigned To... summary location can be identical matches or have a nested hierarchical relationship without presenting a conflict; that is, an asset can be Assigned To... a summary location rack within the Assigned data center. Location Rules ares are applied to tag reports to determine the Detected Location. Detected Location must agree with Assigned To... location within a single hierarchical level.

Manage Equipment Without Assigned Location

 

Click this link to view tagged assets without assigned locations. Double-click an asset from the list to view its attributes. Right-click the asset for more options.

A tagged asset detected in a location to which no rules have been applied may report as Detected in Unknown Location.
