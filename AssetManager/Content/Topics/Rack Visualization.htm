Manage Rack Assets

Racks that have been created as Summary Locations can be managed through a graphical display. The Rack View feature is a grid, two- or three-dimensional representation of a rack and the assets assigned to it. This feature is only available for Summary Locations set as Racks. From the View Rack/Edit Rack tab, you can view or edit:

  • All assets assigned to the rack
  • Which assets have associated tags
  • U-mount assignments for each asset
  • Empty space in racks
  • Rack Issues including placement conflicts and assets detected elsewhere

View Rack

  1. To view rack usage, navigate to User Console > Assets, select any of the views that include racks, and click on a rack from the asset list.

 

  1. Double-click on a rack, right-click and select View, or select the rack and click the View button to view the Rack pop-up window.

 

  1. Select the View Rack tab to view a graphic display of the rack᾽s contents.

 

Click the Grid, 2D, or 3D button to change view options, or the Front/Back toggle to change orientation.

The View Rack tab displays the front of the rack by default and resets to the front if closed, but will retain your last selection of grid, 2D, or 3D.

If you need to view the rack graphic from an angle, choose 3D and then drag the rack to the desired orientation.

Rack items are displayed using the default or user-specified skins. Conflicts in rack position assignments are listed in the Rack Issues task pane and shown in the graphic display.

Edit Rack

  1. To edit the contents of a rack you were viewing, close the View Rack pop-up and right-click on the rack in the asset list, which will be highlighted, then select Edit.
    Or navigate to User Console > location > rack or User Console > Manage Assets > Type > Rack, click on a rack in the Asset list, and click Edit.

 

  1. The Edit Rack tab opens. Click the help icon to view Rack Edit Instructions, or select the item to be edited.

Customize Asset Skins

Image overlays can be used to differentiate the assets mounted in a rack. To edit a single asset’s overlays, navigate to the rack where the asset is assigned and Edit: Rack. Within the Edit Rack tab, select the asset, right-click for menu options, and Change skin.

To add a customized skin for an asset type, navigate to Admin Console > Customization > Asset Skins.

 

Click New to browse to and upload a new image (or image-set for front and rear) and set all details manually, or Copy to add a new skin image with similar details to one of the default skins. Save the skins to the desired folder.
