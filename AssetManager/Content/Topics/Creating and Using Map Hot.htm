Map Hot Spots

After an image has been uploaded, the image appears in the right pane and the Hot Spot Tools control bar appears. Hot Spot Tools allow you to configure interactive elements for your map.

After creating a map, one of the first features to explore is Hot Spots, so you can, for example, create a click-through path to another map. 

 

Hot Spot Tools
1 Select & Edit Move, edit, copy, or delete object. To edit a rectangle or a polygon hot spot shape that you have drawn, click on this tool, then click on the object you wish to edit. The outline of the shape is now highlighted and drawing points are available on all of the corners of the shape. Click and drag any of the points that you would like to move.
2 Draw Rectangle

Create a rectangular hot spot. 

3 Draw Polygon

Create a polygonal hot spot.
4 Place Reference Point

Place a Hot Spot target on the map without creating a containing shape. Useful for items within or on other items, for instance, a sensor tag on an asset, and where the map already has a container displayed and no formatting is required.
5 Duplicate Shape Copy and paste a shape from one you have previously drawn. To duplicate a shape, select the shape and click this button.
6 Place Map Attributes Box Place the Map Attributes box on the map. A Map Attributes Box displays various attributes about the map. The map attributes box will resize in proportion with the image when being viewed.
7 Move Forward Move the item in front of preceding item.
8 Move Backward Move item behind preceding item.
9 Move to Front Move the item in front of all other objects.
10 Move to Back Place item behind all other objects.
11 List Hot Spots

Lists all Hot Spots created. To view the List of Hot spots, select the List Hot Spots tool.
12 Show Hot Spots Info Turns on a rollover function that allows hot spot info to be seen in the Map Editor pane when you rollover the hot spot. The function can be turned on by clicking on the button and turned-off by again clicking on the button.
13 Delete Shape Click the object to be deleted, then click this tool and Save Changes.

Use the second icon from the left to create a rectangular hot spot. 

If you want to create a multi-dimensional hot spot (that is, one that is not rectangular), then click the third icon from the left (the one immediately to the right of the square/rectangle). 

Hot Spot Settings

 

  • Hot Spot Target: Choose the type of target for the hot spot drawn (Asset, Location, Text, or URL). The hot spot attributes that will display in the reference box will relate to the target that is chosen and will also link to that location, asset or URL when the hot spot region is clicked upon in the map in the User Console. This will not be the case for the text target type. The text target type cannot hold attributes or be a hot link.
  • Asset/Location/Text/URL: This field varies depending on the type of hot spot target that is chosen. If a location target is chosen, this field will require you to select a location to scope to from within the tree and will optionally allow you to choose a different map family than the map family currently being viewed. If an asset target is chosen then this field will require you to select an asset from your assets list. If text is chosen, this field allows you to add any text to the map, but will not hold any attributes or be a hot spot. If a URL target is chosen then this field will require you to input a URL to connect to and optionally will allow you to apply a label to this URL.
  • Data Box: Choose between None, Visible, or Include Hover. If Include Hover is chosen, the text that would normally display only when the user hovers the mouse over the hot spot will instead display on the map. (Not available for Text or URL.)
  • Fill Color: Choose a fill color for the hot spot shape that has been drawn. If you do not specify a fill color, this setting will default to the Fill Color setting specified in your map᾽s settings. If your map᾽s settings do not have a fill color specified then the fill color will default to the Fill Color setting found in Global Settings.
  • Fill Opacity: Specify an opacity percentage that you would like for the fill area. The lower the percentage the more transparent the fill area will appear. For example a specification of 100% for this field will make the fill area completely opaque. If you choose not to specify a percentage for this field, this setting will default to the Fill Opacity setting specified in your map᾽s settings. If your map᾽s settings do not have a fill opacity specified then this setting will default to the Fill Opacity setting found in Global Settings.

The width of the hot spot data box will change relative to the scale and/or contents when viewing the map. The size of the data box shown when editing is the “Max width” when viewing the map image at 100%, meaning that depending on what text is being displayed, the data box may appear smaller than the width specified, but it will not grow wider than the width specified.

Use Hot Spots to Nest Maps

Maps are generally separated by function; that is, one map may be made for displaying sensor readings, and a second map made for clicking through from larger to smaller geographical areas. Depending on the amount of information to be displayed, these functions might be bundled into a single map. 

In the screenshot below, a building floor plan is used to provide users a visual means of selecting a desired location. Four Hot Spots have been created, each linking to another map. The Hot Spot Settings pane displays information for the selected Hot Spot, Data Hall 1200.

 

When accessed from the User Console, Hot Spot Settings are displayed.

 

Clicking on the hot spot brings up the nested map. This map displays all racks in Data Center 1200. When viewed from the User Console, the map includes real-time displays of Alerts, and each rack’s Asset Attributes display on hover.

 

To preview a map in a new browser tab, click the View button.

Use Multiple Hot Spot Types on A Map

In this next example, a Hot Spot has been created for a different rack, but instead of allowing a click-through to the individual rack location, this Hot Spot shows the average intake temperature of the rack as a whole.  This is done by pointing to the Summary Location for the rack because, by default, the Summary Location provides an aggregate of all sensor readings associated with the rack. 

View Maps in User Console

  1. Navigate to User Console > Maps and select the appropriate Location from the Location pull-down menu.
  2. Choose a specific Map, View, Attribute, Zoom, Minimum Alert Severity,and/or Attribute Formatting from the pull-down menus. If only a single item is available, the pull-down will be grayed out.

Minimum Alert Severity hides Hot Spots for any objects not meeting the filter criteria. This allows users to see only those items with alerts of the specified level or above.

  1. Hover over a Hot Spot to view information about a Location or Asset, if this option has not been disabled by the System Administrator for this Map View.
