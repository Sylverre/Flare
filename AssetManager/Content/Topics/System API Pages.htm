System API Pages

In version or greater, the System Administrator has the capability to access a set of System API pages for diagnostic purposes. These pages can be accessed by inputting a simple URL:

http://<HOSTNAME>:6580/api/system/<NAME OF SYSTEM API PAGE>

Files

This System API page contains links to all our log files. The page has a button to download a zip file containing the logs.

Log

This System API page allows you to change log levels for all the different components of . This is transient, a restart resets the log levels.

Property

This System API page lists the system properties.

Stat

This System API page lists various statistics that are tracked by the system (eg. tag event rates) threads. It lists all the threads and their stack traces.
