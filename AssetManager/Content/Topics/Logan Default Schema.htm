Data Schema

includes a data schema containing asset types and attributes that allow you to control the categories of assets and sensors you monitor and the specific attributes to track.  This schema is designed with the data center in mind and contains asset types and attributes suited to monitoring temperature, humidity, air pressure, power, etc. in your deployment environment.  It also contains common inventory asset types and attributes suitable for managing the assets in data centers.

To view the specific asset types, attributes, etc. contained within the schema, navigate to Admin Console > Data Schema.  Click a schema item to view its details. Click New to add a new attribute manually, or Admin Console > Configuration > Import Configuration to import a new schema from CSV.
