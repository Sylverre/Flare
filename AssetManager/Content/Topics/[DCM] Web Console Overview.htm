Web Interface Overview

The web interface contains a navigation menu on the left with links and sub-links, or Tasks and Sub-Tasks that you click in order to populate one or more panes of information on the right side.  These panes will either simply display information about various parts of the system or they will contain additional fields or functions that you can configure.

web interface is arranged with a toolbar across the top, navigation panel on the left, task panes on the right, and an information bar along the bottom of the screen.

1 Toolbar -- quick-access buttons reflect allowable actions within the view.
2 Navigation Pane --
3 Task Pane(s) -- one or more panes
4 Information Bar -- displays information about user and alerts. Right corner contains toggle, about, link, and help buttons.

There are two primary consoles (views) in :

  • Admin Console: Access this console by clicking the Admin Console link in the lower right corner of the web interface of . This console is typically used for administrative functions of the server and where a large portion of this document focuses. It could be considered the “Control Panel” of your . This is where most of the initial configuration is done.
  • User Console: Access this console by clicking the User Console link in the lower right corner of the web interface. This console is for managing and viewing sensors, alerts, dashboards, reports, graphs, etc. This is where the final configuration and daily use of is done.

NOTE: Switching between the Admin Console and the User Console is done by clicking either the Admin Console or the User Console link at the bottom right of the user interface. The name of the link changes depending on which console you are using and will always display the name of the console that you are not currently using.
