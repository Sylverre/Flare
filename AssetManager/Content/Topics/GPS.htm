GPS

This sub-task enables the user to adjust the global GPS settings for all GPS devices that are connected to the system.

  1. Select and input desired global GPS settings. Click the Save Changes button to save the settings.
Global GPS Defaults: GPS Data Reporting
  • LAT - Latitude
  • LON - Longitude
  • EPE - Estimated Position Error
  • SPD - Speed
  • ALT - Altitude
Global GPS Defaults: Minimum GPS Update Period
  • Input the minimum time period required for GPS data updates. The default is set for 30 seconds.
Global GPS Defaults: Minimum Horizontal Change
  • Input the minimum distance required for horizontal change for GPS data updates. The default setting is 10 meters (32.8 feet).
Global GPS Defaults: Minimum Vertical Change
  • Input the minimum distance required for vertical change for GPS data updates. The default setting is 100 meters (328.1 feet).
