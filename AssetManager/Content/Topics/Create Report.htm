Create a New Report

Reports can be created from either the Admin Console or the User Console. Reports about the assets and environments you are managing and monitoring are accessed through the User Console, while reports about the system and user access are managed in the Admin Console.

  1. Navigate to Admin Console > Reports/Graphs or User Console > Reports/Graphs and click the New button.
  2. From the System Report drop-down, select the type of report to create.
  3. Complete all required and any desired additional fields.
