  1. Open by double-clicking the shortcut that was installed on your desktop or navigate to Start> Programs> RF Code> or navigate with your browser to http://<server host name>:6580. The following screen will appear. In order to begin the initial setup of the software, you must log in with the default Administrator user name and password which is as follows:

username: admin

password: admin

Please note that the login is case sensitive.

Figure 19: Initial Screen

  1. Database Configuration: requires a database to function. All configuration and information is stored in this database. If you did not choose to install Microsft SQL Server Express 2008 as part of the installation process, you will be automatically directed to the database configuration screen. requires access to a database with full read/write privileges. If you are using a remote database, then the database administrator must create the database and create a userid/password that has full access to the database. If you are using a local copy of Microsoft SQL Server Express, these items were setup at installation time. Please enter your database information into the fields and click on the Test button.
  1. If you have setup your database correctly, you will receive a message confirming that the database test was successful. Click Save Changes to continue.
  2. A message indicating that the Service needs to be restarted will appear, click OK to continue. You will be redirected to the main login screen again. Sign into again with the default administrator user-name and password.
  3. You should now see the default screen of also known as the Dashboard. Notice that there is a window in the top right corner of the dashboard called System Configuration. At this step it should indicate that your database is correctly configured as well as other required tasks that are not configured. These tasks are the minimal steps required to get into a functional state.

  4. Zone Manager(s) Configuration: Once the database is installed, notice that in the System Configuration Task Pane that your Zone Manager is configured as well. automatically configures a local Zone Manager. In some cases this may be your only Zone Manager. However, in larger deployment scenarios, you may choose to setup other “remote Zone Managers”.
