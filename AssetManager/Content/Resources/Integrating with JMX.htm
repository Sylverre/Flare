Integrating with JMX

JMX is part of the Oracle Java SE Platform and provides a simple, standard way of managing resources such as applications, devices, and services.  The JMX specification defines the architecture, design patterns, APIs, and services in the Java programming language for management and monitoring of applications and networks. For more on JMX, refer to: http://www.oracle.com/technetwork/java/javase/tech/javamanagement-140525.html
