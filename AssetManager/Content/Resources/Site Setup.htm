Site Setup and Schemas in Site Setup and Schemas

After completing the basic configuration, you must set up the system to manage and monitor your assets in your locations.  A Location Hierarchy defines all the locations where assets are located, Asset Types group your assets for management, and Attributes define those aspects of assets and environments for which data will be collected.

A Data Schema Data Schema is included with . This schema includes Asset Types and Attributes suited for managing data center equipment and monitoring data center environmental conditions.

You must create a location hierarchy that includes all of your locations with environments to be monitored and/or assets to be managed.
